Remote Start

Starts a new Live Activity on the device without the app being open. Requires the push-to-start token from the iOS app.

Direct Push

Update or end a specific Live Activity using its push token. Copy the activity update token from the iOS app after starting an activity.

Broadcast Update

iOS 17.2+

Send an update via an APNS broadcast channel. All subscribed Live Activities update simultaneously — no individual tokens needed.
